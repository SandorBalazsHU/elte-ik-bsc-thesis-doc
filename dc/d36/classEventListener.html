◆ getClickedObjectId()

int EventListener::getClickedObjectId ( SDL_MouseButtonEvent &  mouse,
bool  vehicleCheck = false 
)
private

Ray tracing mouse picking implementation.


Calculating a ray true the camera and the mouse picked point and test the intersections for all the hit spheres.
The vector generation is a revers incremental image synthesis.
When have multiple hit sphere in one line we select the one closest to the camera.
The function's main step's:
Step 0: Get the screen size.
Step 1: Get the 3d Normalized mouse coordinates in Device Coordinates.
Step 2: Convert the device mouse vector to 4d Homogeneous Clip Coordinates.
Step 5: Normalize the world mouse vector.
Step 6: Intersect the world mouse vector with all the renderable object's hit spheres.
Step 7: Depth test.

Parameters
mouseThe current mouse click event.
Returns
The selected item render id.
